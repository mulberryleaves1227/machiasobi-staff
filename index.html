藍場浜 撮禁A / 撮禁B
AとBの2名1組で交代制
🔁 A/B交代制のルール▼
  • 撮禁Aの番：A が札を持って巡回、B は待機・休憩
  • 撮禁Bの番：B が担当、A は待機・休憩
  • 交代タイミングはシフト表またはリーダーの指示に従う
📋 撮影禁止対応の手順▼
  • 撮禁札・表示を来場者から見える位置に出す（観覧者の視界をふさがない高さ・位置に注意）
  • ステージ正面だけでなく横・後方・橋の上・対岸・歩道側も確認する
  • スマホ・カメラを構えている人に落ち着いて声をかける
  • 注意しても続ける場合はリーダー・八木・桑野へ共有
  • 30分以上立ち続けることがある — 体調に注意
🚫 やってはいけないこと▼
  • 来場者のスマホ・カメラに触れる
  • 撮影データの削除を個人判断で求める
  • SNS投稿の削除を個人判断で求める
  • 強い口調で注意する・口論する
📍 藍場浜エリア 見るポイント▼
  • ステージ後方・横導線で人が止まっていないか
  • 通路がふさがっていないか
  • 終演後に一斉退場で詰まっていないか
  • 次のステージ待機列と退場導線がぶつかっていないか
  • ステージ上・出演者の撮影が行われていないか
  • 体調が悪そうな人がいないか

立ち上げ・撤収
🔧 立ち上げでやること▼
  • ステージ周辺の三角コーン・バーの展開
  • 撮影禁止札・案内表示などの設置
  • パラソル各ブースへの看板・配布物の配布
  • 出展者・ブース担当者への挨拶（当日の連携づくり）
  • ブース位置、導線、問い合わせが起きそうな場所の確認
  • ごみ捨て場所、パス受け取り場所、本部テントの案内
  • 不明点は本部・混雑対応部・八木・桑野へ確認
📦 撤収でやること▼
  • 三角コーン・バー・案内表示の回収
  • 撮影禁止札・備品の回収
  • パラソル周辺の忘れ物・ごみ確認
  • ブース担当者への終了確認・挨拶
  • 残置物や破損物の確認
  • 余った配布物・看板等の回収先確認
  • 本部への完了報告
ステージイベント 会場整理誘導
📋 主な業務▼
  • 来場者の安全を確保する
  • 観覧エリア・通路・待機列を整理する
  • 撮影禁止や観覧ルールを守ってもらう
  • 終演後の混雑を防ぐ
👥 2人1組での行動▼
片方が来場者対応をしている間、もう片方は周囲の安全確認・通路確保・リーダーへの連絡を担当。2人とも目の前の対応に入り込みすぎない。

すぐ共有するケース：
  • 強い口調で詰められている
  • 列が伸びて通路をふさぎそう
  • 撮影禁止への注意に応じてもらえない
  • 体調不良者・迷子・クレームが発生した
受付（藍場浜 / 東公園 / 西公園）
📋 テントで受ける可能性がある対応▼
  • 会場・イベント場所の案内
  • 落とし物・迷子・同行者不明の一次受付
  • 体調不良・怪我の初動共有
  • アンケート記念品の案内・受け渡し
  • スタッフやリーダーへの連絡・呼び戻し
  • 関係者・出展者からの問い合わせの一次受け
⚠️ パス類・関係者対応は単独判断NG▼
スタッフパス・関係者パス・取材証・出展者証などの問い合わせは、必ず混雑対応部へエスカレーション。近くにいない場合は八木・桑野へ。

使いやすい一言：
「パスや関係者受付に関する内容は確認が必要です。少々お待ちください」
遊撃 / BW（ボードウォーク）
会場内の「困った」に素早く対応するフリーポジション
📋 主な役割▼
  • 急に伸びた列の列形成支援・最後尾案内
  • 通路確保、立ち止まり・滞留への声かけ
  • ステージ終了後や物販開始前後の誘導補助
  • 駐輪場・コルネの泉周辺の巡回・通行確保
  • 本部・テント・現場間の連絡補助
  • 備品・ごみ袋・記念品などの運搬補助
  • 迷子・体調不良・落とし物の初動共有
  • 「なんか詰まりそう」という違和感を拾って共有する
💬 共有例▼
【遊撃】ポッポ街商店街、ブース前の列が通路中央にはみ出しています。最後尾がわかりにくく、通行と混ざり始めています。列位置の判断と追加スタッフが必要です。

【遊撃】藍場浜、終演後の退場者と次イベント待機が交差しそうです。後方通路の確保に入っています。導線判断をお願いします。
⚠️ 安全を最優先に▼
慌てて走らない。車道・水辺・階段付近での無理な移動をしない。指示が重なった・シフト時間を超えそうな場合は、必ずリーダー・八木・桑野へ確認。
自転車置場〜東新町 / コルネの泉 巡回
📋 主な役割▼
  • 駐輪場周辺・東新町・コルネの泉周辺を巡回する
  • 通路や店舗入口をふさぐ滞留がないか確認する
  • サイン会エリア等で待機列ができそうな場合、早めに共有
  • 最後尾・列名がわかりにくい場合、一時的な声かけを行う
  • 体調不良者・落とし物・ごみ・危険な滞留がないか確認する
✅ 対応してよいこと / ❌ 確認が必要なこと▼
対応OK：
  • 通路確保のための声かけ
  • 「何の列か」を確認するための一次案内
  • 混雑状況・最後尾不明・通行支障の共有
要確認：
  • 自転車の移動・撤去・駐輪可否
  • サイン会の参加可否、整列開始、列打ち切り
  • 整理券・参加条件・本人確認

📋 報告テンプレート（コピーして使う）
🔴 混雑報告
【混雑】 場所： 状況：通路が詰まり始めています / 列が伸びています / 最後尾が不明瞭です 人数感： 対応中：声かけ中 / 最後尾案内中 / 通路確保中 相談：追加スタッフが必要です / 導線変更の判断をお願いします
🟠 体調不良・怪我報告
【体調不良・怪我】 場所： 対象者：大人 / 子ども / 不明 状況：座り込んでいる / めまい / 吐き気 / 熱中症疑い / 怪我 同行者：あり / なし / 不明 対応中：日陰へ案内 / 安全な場所で待機 / 救護待ち 相談：救護担当の派遣をお願いします
📷 撮影トラブル報告
【撮影トラブル】 場所： 内容：撮影禁止エリアで撮影 / 無断撮影の相談 / 通路をふさぐ撮影 関係者：撮影者 / 被写体 / 周囲の来場者 対応中：撮影停止のお願い / 場所移動のお願い / 確認中と案内 相談：リーダー対応をお願いします
👦 迷子・同行者不明報告
【迷子・同行者不明】 場所： 対象者：子ども / 大人 / 不明 状況：泣いている / 困っている様子 / 探している来場者あり 特徴（わかる範囲）：服装・年齢・名前等 対応中：安全な場所で待機 / 確認中と案内 相談：テントへの誘導・放送対応の判断をお願いします
🎒 落とし物報告
【落とし物】 場所（拾った場所・時間）： 物品（わかる範囲で）： 持ち主：あり（いる）/ なし / 不明 対応中：本人待機 / テントへ案内中 / 預かり中 相談：テントへの引き渡し確認をお願いします
😠 クレーム・強い申し出報告
【クレーム】 場所： 内容（聞いた内容の要点）： 相手の様子：落ち着いている / 強い口調 / 長時間対応中 対応中：話を聞いている / 確認すると伝えて保留中 相談：リーダー対応をお願いします / 対応方針を教えてください
